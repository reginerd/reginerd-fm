What is this?

WRIT-FM is an experimental internet radio station that runs entirely on AI. An LLM writes DJ scripts, text-to-speech voices read them, and a gapless streamer mixes talk segments with curated music — all day, every day.

How it works

  • A schedule of themed shows rotates through the day — philosophy at midnight, music history in the morning, current events at noon.
  • An LLM (Claude CLI or Gemini CLI) writes DJ monologues, intros, weather reports, and listener responses in each show's voice.
  • Kokoro TTS renders the scripts to audio with 28 built-in voices.
  • An ezstream + feeder.py pipeline builds per-show playlists, interleaves talk segments with AI-generated music bumpers, and pipes everything to Icecast.
  • An operator daemon (Claude Code on a 15-minute loop) keeps content stocked, health-checks the stream, and carries editorial continuity across runs.

Live Streams

The station runtime can host multiple live streams at once. Each stream has its own queue, API port, operator state, and Icecast mount.
